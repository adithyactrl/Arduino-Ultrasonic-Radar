# 📡 Sensor Radar Web Dashboard(This interface was created with AI assistance as a starting point for understanding sensor visualization and web-based hardware monitoring)

A web-based visualization interface designed to display live sensor readings and represent object movement using a radar-style display.

## ⚙️ Overview

This project acts as a visual layer for the sensor-based system.

The webpage receives sensor data and converts it into a graphical radar interface, helping visualize object detection and movement in real time.

## ✨ Features

- 📡 Real-time sensor data visualization
- 🟢 Radar-style object detection display
- 📊 Live distance/value updates
- 🖥️ Browser-based interface (HTML/CSS/JavaScript)
- 🔄 Dynamic movement representation

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
